1. сериализация - преобразование какого либо объекта в поток данных (байтов). десериализация обратный процесс, логично).
2. 1) Создание экземпляра потока  
   2) Открытие потока (для чтения или записи)  
   3) чтение из потока / запись в поток
   4) Закрытие потока
3. InputStream — поток для чтения байтов (поток ввода)
   Reader — поток для чтения символов (поток ввода)
   OutputStream — поток для записи байтов (поток вывода)
   Writer — поток для записи символов (поток вывода)
4. IO - потокоориентированный, синхронный ввод/вывод
   NIO - буферориентированный, ассинхронный ввод/вывод
NIO позволяет программе продолжать выполнение других задач, не ожидая завершения операции ввода-вывода.
